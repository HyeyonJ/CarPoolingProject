## Carpooling(카풀 - 링)
추운 겨울, 더운 여름 택시를 잡으려고 했지만 어려움을 겪으셨나요?<br>
"carpooling(카풀링)"은 경로를 탐색하여 운전자와 사용자를 매칭하는 카풀 서비스입니다.<br><br>
